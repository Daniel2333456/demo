DREAM REEL - PURE FRONT-END DEMO

FILES
-----
Home.html          Home page
discover.html      Search and filter page
player.html        Video player and episode page
favorites.html     Favorite dramas page
profile.html       Personal center page
css/style.css      Shared responsive styles
js/data.js         Drama and episode data
js/app.js          All front-end interactions

HOW TO RUN
----------
1. Open the entire folder in Visual Studio Code.
2. Install the "Live Server" extension.
3. Right-click Home.html.
4. Choose "Open with Live Server".

The project has no back end and no database.
Profile, favorites, theme, membership and watch history are stored only in browser localStorage.

REPLACE IMAGES OR VIDEOS
------------------------
Edit js/data.js.

Replace a poster URL:
poster: "images/your-poster.jpg"

Replace a video URL:
video: "videos/episode-1.mp4"

You may create local "images" and "videos" folders and place your own files there.
